> [!NOTE]
> 在原 clash-verge-rev 相同的数字版本（没有`-patch`后缀）的基础上，添加的一些功能和改进

### ✨ Features
- 连接界面，服务模式界面的 i18n 完善
- 连接页面添加字段显示、字段内容条件过滤功能按钮
- 系统设置的代理绕过为空时，使用默认的代理绕过规则
- 支持 Windows 和 Linux 下是否启用系统标题栏设置
- 窗口最大化后取消圆角
- 左侧菜单栏选中时图标高亮
- 左侧菜单栏的涟漪效果颜色使用系统主题色
- 支持UI界面是否保持活跃状态设置
- 使用 monaco 编辑器编辑主题设置的 CSS 样式注入
- 代理组界面添加右侧边栏展开显示代理组列表，实现点击跳转
- 配置文件新建/编辑的文件类型使用更醒目的按钮组
- 给重新激活配置文件，启用/关闭/删除增强文件(merge/script)操作添加动画
- 支持同种状态下（启用/未启用两种状态）的增强文件/增强脚本进行拖拽排序

### 🐛 Bug Fixes
- 代理组过滤节点按钮关闭后未重置过滤信息
- Linux 开发环境下代理组图标加载失败
- **尝试修复** 重启内核导致的流量、使用内存、日志数据丢失，界面显示异常
- Linux 系统设置的代理绕过为空时显示错误
- alpha 内核更新升级，已经使用最新版本的提示信息类型为错误类型
- 窗口尺寸变化导致 monaco 编辑器无法填满内容、小地图功能不生效
- 窗口尺寸频繁改变导致代理页面布局变化卡顿
- 窗口销毁重新创建后第一时间渲染的主题不是用户配置的主题
- 窗口的创建在 Linux 中不支持 set_shadow 方法
- 剪贴板粘贴按钮无效
- 配置文件/增强文件/增强脚本启用失败时无报错信息提示
